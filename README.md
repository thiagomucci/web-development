# Web Development Projects

Este repositório contém projetos de front-end criados para estudo e prática de **HTML, CSS e JavaScript**, com foco em DOM, eventos e interatividade, os projetos incluem

- Contador interativo com botões de incrementar, decrementar e resetar  
- Dark Mode com switch animado de lua, estrelas e nuvem  
- Formulário com validação de campos vazios e realce visual

---

## ⚙️ Funcionalidades

### Contador
- Incrementa, decrementa e reseta valores  
- Atualização dinâmica do valor usando `innerText`  
- Botões com efeitos de clique

### Dark Mode
- Alterna entre modo claro e escuro  
- Switch animado com lua, estrelas e nuvem
- Transições suaves de cores usando CSS

### Formulário
- Campos de texto, senha e email.  
- Validação ao clicar em enviar: campos vazios recebem borda ou contorno vermelho
- Uso de `classList.add/remove` para indicar estado do input 
- Transições suaves de cor para realce visual


## 💻 Tecnologias utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilos, transições, animações e design responsivo  
- **JavaScript (ES6+)** – Manipulação do DOM, eventos e lógica interativa  
- **Flexbox** – Layout centralizado
